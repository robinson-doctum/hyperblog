Comando que he usado

git add "nombre del archivo"
    añade el al¡rchivo al commit, queda en el estado de espera ara ser enviado

git rm "nombre del archivo"
    elimina el archivo

git status 
    muestra el estado que se encuentra el archivo

git config
    ve la configuraciones globales

git config --list 
    muestra las configuraciones del git "user y demas"

git commit -"mensaje"
    enviar el commit al repositorio, se concidera buena practica enviar el mensaje del commit

git show "nombre del archivo"
    muestra el historial de commit del archivo

git log "nombre del archivo"
    muestra los cambios del archivo

git diff "codigo de commit" "codigo de commit"
    sirve para compara los cambios realizados entre dos commit, el codigo del commit se obtienes con el show

git reset "codigo de commit" --soft "indica que el cambio va acurrir en el status" --hard "el cambio va ocurrir en el repositorio"

para realizar un commit se veria seguir estos pasos
1. ejecutar git add "nombre del archivo" , en el caso que se desea que se mande todos los archivos de las carpeta se puede ejecutar add . , el punto indicar que son todos los archivos
2. se ejecutar luego el git commit-me"el ensaqje es buena practica"
